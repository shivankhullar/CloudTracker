## Installation

To install CloudTracker, follow these steps:

1. **Clone the repository**:
    ```sh
    git clone https://github.com/yourusername/CloudTracker.git
    cd CloudTracker
    ```

2. **Build the project**:
    ```sh
    make 
    ```
